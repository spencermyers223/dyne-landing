# Dyne Landing Page

The smarter way to dyne. A reservation concierge platform.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Deployment

This project is configured for Vercel deployment. Connect the GitHub repo to Vercel and it will auto-deploy.
